######  Как запустить

- set your open weather key in `config.json`
- `docker-compose build && docker-compose up`


###### Описание решения
- Умный сервис прогноза погоды (уровень 3), узнает погоду за окном для введенного города, веселит гифками и напоминает, что нужно оставаться дома 
- Веб-сервер на java + фронтенд на бутстрапе.
- MemWeatherAnalyzer с точки зрения реализации тоже самое что и подбор одежды. Достаточно реализовать имплементацию ClothingWeatherAnalyzer для интерфейса WeatherAnalyzer
- В зависимости от текущей погоды за окном формирует человекопонятное описание, а также подбирает гифку, которая илюстрирует описание

###### Описание работы
- → пользователь вводит город и страну
- → веб-сервер отправляет запрос за текущей погодой и анализирует данные
- → отправляет ответ на клиент
- → клиент отображает данные пользователю
